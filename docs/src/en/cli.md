# nunu命令行工具 {#info}
nunu 命令行工具是一个为了协助快速开发 Golang 项目而创建的项目，通过 nunu 命令行工具， 您可以很容易的进行 Golang 项目的创建、热编译、开发、测试、和部署。

## 创建项目 {#new}
```
nunu new [projectName] 
```
## 启动项目 {#run}
```
nunu run
```

## 创建组件 {#create}
```
nunu create [type] [handler-name]
```
### 创建handler
```
nunu create handler [handler-name]
```
## 编译wire {#waire}
```
nunu wire all
```

## 版本升级 {#upgrade}
```
nunu upgrade
```
